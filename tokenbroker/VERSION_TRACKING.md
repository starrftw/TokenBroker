# TokenBroker Version

## Overview
TokenBroker maintains a single version that applies to both the agent and skill (they are the same product).

## Current Version: 1.0.1

## Where to Find Version
- **All Metadata**: `tokenbroker/VERSION` file
- **Skill Frontmatter**: `tokenbroker/SKILL.md` YAML frontmatter
- **Changelog**: `tokenbroker/CHANGELOG.md`

## Version Format
Semantic Versioning (MAJOR.MINOR.PATCH):
- MAJOR: Breaking changes to agent/skill architecture
- MINOR: New features or significant updates
- PATCH: Bug fixes and minor improvements

## Update Checklist
When updating version:
1. Update `tokenbroker/VERSION` file
2. Update `SKILL.md` frontmatter
3. Add entry to `tokenbroker/CHANGELOG.md`
